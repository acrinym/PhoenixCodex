================================================================================
                          QUICK START
================================================================================

Convert one chat to readable Markdown:
  1. python chatgpt_converter_gui.py
  2. Tab: "🔄 Converter"
  3. Browse to JSON file
  4. Click "🚀 Convert"
  5. Open output .md file

Convert entire folder:
  1. python chatgpt_converter_gui.py
  2. Tab: "📦 Batch"
  3. Input: Your chats folder
  4. Output: Backup folder
  5. Click "🚀 Start Batch"
  6. Wait for completion


================================================================================
                          WHAT EACH TOOL DOES
================================================================================

Converter GUI:
  Take ChatGPT JSON files and convert them to readable Markdown
  → Or convert Markdown back to JSON
  → Can filter by role (user/assistant)
  → Can process entire folders at once
  → Simple, fast, no configuration needed

Export Tool:
  Search across all your ChatGPT exports
  → Build indexes for fast searching
  → Export to multiple formats
  → Classify content automatically
  → Advanced features: semantic search, context extraction

Dataset Builder:
  Extract and analyze patterns in your data
  → Performance optimization
  → Memory management
  → Batch processing capabilities
