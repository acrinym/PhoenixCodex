INSTALLATION
============

No installation needed! Just run:

    python chatgpt_converter_gui.py

Requirements: Python 3.7+, tkinter (included with Python)


USAGE - THREE EASY STEPS
========================

1. SELECT YOUR MODE
   ├─ 🔄 Converter Tab: Convert one file at a time
   ├─ 📦 Batch Tab: Convert an entire folder
   └─ 👁️ Preview Tab: View/inspect content

2. CHOOSE YOUR FORMAT
   ├─ JSON (native ChatGPT format)
   └─ Markdown (readable text format)

3. CLICK THE BUTTON AND WAIT


COMMON TASKS
============

Task 1: Convert One Chat to Readable Markdown
──────────────────────────────────────────────
1. Open chatgpt_converter_gui.py
2. Click Tab: "🔄 Converter"
3. Input File: Browse to your JSON file
4. Output Dir: Choose where to save
5. Input Format: json
6. Output Format: md
7. Click "🚀 Convert"
8. Open the .md file in any text editor

Time: 1 minute


Task 2: Convert All Chats at Once
──────────────────────────────────
1. Open chatgpt_converter_gui.py
2. Click Tab: "📦 Batch"
3. Input Folder: D:\Chatgpt\ExportedChats\chatgpt-export-json
4. Output Folder: C:\Users\YourName\Downloads\chats_markdown
5. Output Format: md
6. Check: "Recursive" (to include all subfolders)
7. Click "🚀 Start Batch"
8. Let it process (shows progress)

Time: 5-30 minutes (depending on file count)
Result: All your chats as readable .md files


Task 3: Extract Only Your Questions from a Chat
────────────────────────────────────────────────
1. Open chatgpt_converter_gui.py
2. Click Tab: "🔄 Converter"
3. Input File: Your chat JSON
4. Filter Messages: User (only your messages)
5. Output Format: md
6. Click "🚀 Convert"
7. Result: Markdown with only your prompts

Time: 1 minute


WHAT IT DOES
============

INPUT: ChatGPT JSON Export
├─ Who said what
├─ When it was said
└─ The full conversation

OUTPUT: Readable Markdown (optional JSON)
├─ Pretty formatted text
├─ Easy to read and search
├─ Can filter by speaker
└─ Can include timestamps

TIPS & TRICKS
=============

✓ Backup originals before batch processing
✓ Use Markdown for reading, JSON for processing
✓ Filter by role to isolate specific speakers
✓ Batch process overnight for large folders
✓ Use Preview tab to inspect before conversion
✓ Check the log for any errors


TROUBLESHOOTING
===============

Problem: "Input file not found"
→ Check the file path is correct
→ Try using the Browse button instead

Problem: Output looks garbled
→ Make sure text editor supports UTF-8
→ Try opening in VS Code

Problem: Very slow on large files
→ Large files (>100MB) slow down the process
→ Convert smaller batches instead

Problem: No output file created
→ Check the status log for errors
→ Verify output folder has write permissions
→ Try different output folder


FOR PYTHON PROGRAMMERS
======================

Import and use in your code:

    from chatgpt_converter_gui import ChatGPTExportConverter
    from pathlib import Path
    
    converter = ChatGPTExportConverter()
    
    # Parse JSON
    data = converter.parse_chatgpt_json(Path('chat.json'))
    messages = converter.extract_messages(data)
    
    # Convert to Markdown
    md = converter.messages_to_markdown("Title", messages)
    
    # Convert to JSON
    json_out = converter.messages_to_json("Title", messages)
    
    # Filter messages
    filtered = converter.apply_filter(messages, "user")
